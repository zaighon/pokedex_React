# Pokédex con React y TypeScript

Este proyecto hace parte de mi proceso de formación como desarrollador frontend.  
Fue desarrollado con el objetivo de practicar React, TypeScript y el consumo de APIs REST.

## Tecnologías utilizadas
- React
- TypeScript
- Vite
- PokéAPI

## Funcionalidades
- Listado de Pokémon
- Búsqueda por nombre
- Consumo de datos desde una API externa

## Estado del proyecto
Proyecto en desarrollo y mejora continua a medida que avanzo en mi capacitación.
